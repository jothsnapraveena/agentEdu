Part 1

From tools to agents — what changed?

You've used AI. You've typed a question into ChatGPT and gotten an answer. That's useful — but it's not what's reshaping industries right now. The thing reshaping industries is agentic AI.

Here's the difference. A tool does one thing when you ask it to. A calculator computes. A search engine retrieves. A basic chatbot answers. You push, it responds, done.

An AI agent is different. Give it a goal, not a command — and it takes a sequence of actions to achieve it. It plans, acts, observes what happened, adjusts, and tries again. It doesn't stop after one response. It keeps going until the job is done — or until it decides it can't do it.

Key Definition

An AI agent is a software system that perceives its environment, reasons about a goal, takes actions using tools, observes the results, and adapts — repeating until the goal is reached or it gives up.

The shift from "AI that answers questions" to "AI that completes tasks" is the shift from tools to agents. It's why companies are hiring AI engineers faster than they can find them. And it's why you're here.

Part 2

The reasoning loop: Think → Act → Observe

Every AI agent — from the simplest research bot to the most complex enterprise pipeline — runs on the same fundamental loop. Understanding this loop is the foundation of everything in this course.

The Agent Reasoning Loop

Every agent repeats this cycle until it reaches its goal

THINK
Reason & plan
what to do next
→
ACT
Use a tool or
take an action
→
OBSERVE
See what happened
and update memory
→
THINK
Repeat until
goal is reached

Let's trace through a real example. Say you tell a research agent: "Find me everything published about KEDA autoscaling in 2025."

THINK: "I need to search for academic papers and blog posts about KEDA autoscaling from 2025. I'll start with a web search."
ACT: Calls the web search tool with the query "KEDA autoscaling 2025"
OBSERVE: Gets 10 results back. Three are relevant, seven are not.
THINK: "I have 3 good sources. I should read each one fully and also try a more specific search."
ACT: Opens the 3 URLs. Calls search again with a refined query.
OBSERVE: Now has 5 good sources with full content.
THINK: "I have enough. I'll synthesize this into a summary."
ACT: Writes the summary.
OBSERVE: Done. Goal reached.

That whole sequence happened without you directing each step. You gave it a goal. It figured out how to get there.

# Week 1: AgentEdu — Simulating the Think → Act → Observe loop

def simple_agent(goal, tools, max_steps=5):
    """A minimal agent loop in Python."""
    memory = []
    step = 0

    while step < max_steps:
        # THINK: Decide what to do next
        thought = think(goal, memory)
        print(f"💭 THINK: {thought}")

        # ACT: Use a tool
        action, result = act(thought, tools)
        print(f"⚡  ACT: {action}")

        # OBSERVE: See what happened
        observation = observe(result)
        memory.append(observation)
        print(f"👁  OBSERVE: {observation}\n")

        if goal_reached(result):
            break
        step += 1

    return memory

# Run it:
simple_agent(goal="Find 3 facts about AI agents", tools=["search", "summarize"])
